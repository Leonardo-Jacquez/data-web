# Data Web INDEX

## How to Add New Information to the Data Web

1. Perform research using semantic and keyword searches.
2. Identify valuable concepts.
3. Follow Single Canonical Copy Rule: Only one definitive version of each piece of information.
4. Create/update files in `research/` or `patterns/`.
5. Update INDEX.md with entry, description, wikilinks, Source-of-Truth Chains.
6. Use proper markdown and references.
